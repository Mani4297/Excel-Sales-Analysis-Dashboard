# Interactive Sales Analysis Dashboard

## 📊 Project Overview
This project involves transforming raw sales data into a professional, interactive business dashboard. It provides real-time insights into product performance, regional sales distribution, and salesperson KPIs.

## 🛠️ Features
- **Dynamic Filtering:** Integrated Slicers with Report Connections to filter all charts simultaneously.
- **Data Modeling:** Used PivotTables to aggregate and structure raw sales data.
- **Visualizations:** Bar Chart (Product Sales), Pie Chart (Regional Share), and Column Chart (Salesperson Performance).
